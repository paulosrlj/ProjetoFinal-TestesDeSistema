# ProjetoFinal-TestesDeSistema
Implementação de 2 testes de sistemas para a disciplina Testes de Softwares.



## :heavy_check_mark: Como executar


## Instalar as dependências
:triangular_flag_on_post: <b>yarn</b> ou <b>npm i</b>

## Setar as váriaveis de ambiente no arquivo .env

<p>MONGO_HOST</p>
<p>MONGO_PORT</p>
<p>MONGO_DATABASE</p>

## Executar

:triangular_flag_on_post: <b>yarn dev</b> ou <b>npm run dev</b>

## Executar os testes de sistema

:heavy_exclamation_mark: É necessário o Chrome na versão 90 ou posterior <br>
:heavy_exclamation_mark: É necessário o webdriver do Chrome <br><br>
<code>
sudo apt install chromedriver
</code>

:triangular_flag_on_post: <b>yarn test</b> ou <b>npm test</b>
